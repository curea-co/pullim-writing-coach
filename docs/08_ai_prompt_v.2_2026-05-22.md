## 0. 왜 v0.2인가

v0.1은 재현성·스키마·FIX_COUNT는 통과했으나 **상위 구간(74+)을 ~62로 과압축**(09 §2.2: B −24·E −13·A −12)했다. P-1.1 결정(docs/13 = 대안 A: EPO 분포가 정답)에 따라, 모델 점수를 **EPO 분포로 상향**하는 캘리브레이션을 가한다.

### 1.1 anchor 범위 변경 이력 (P1.1 결과 반영)
- 1차: hold-out 3+2 — anchor D·C·B / 시험 A·E (eval 누수 방지, 일반화 측정).
- 결과(09 v.2): 숨긴 A(68, −6)·E(75, −10)가 EPO ±3 **미달** → 3-shot이 anchor엔 박히되 novel 고점엔 ~6~10점 보수 잔여 편향.
- 결정(a): **A·E를 anchor로 승격(5건 전부).** 데모 5종은 라이브에서도 EPO와 일치. 단 25건 재검증은 회로적이 되므로 **일반화는 P2.4(임의 글)·P5(사용자 테스트)로 이관**.
