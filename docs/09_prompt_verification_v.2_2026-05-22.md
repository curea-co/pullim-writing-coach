## 4. 결정 — A·E anchor 승격 (대안 a)

- **(a) 채택**: A·E를 anchor로 승격 → 5건 전부 anchor(`ANCHOR_IDS=["d","c","a","e","b"]`).
- 귀결: 데모 5종은 라이브 재채점서도 EPO와 일치(쇼케이스 정합). 단 **25건 재검증은 회로적**(anchor=시험셋)이 되어 일반화 신호를 못 줌.
- **일반화 검증 이관**: P2.4(BE 자체 테스트 — 5종 + 임의 글) · P5(사용자 테스트, 실제 학생 글). WBS §7 "최적화는 배포 후 트랙"과 정합.
- (b) 추가 캘리브 기각: 다른 고점 라벨이 없어 A·E에 문구를 맞추면 hold-out이 무의미해지는 과적합.
