## 1. 한 문장 전략 + 결정 척추

> **"한국에서 유일하게, 표시광고법을 통과하는 방식으로 '학생이 직접 쓴 과정'을 증빙하고 '글쓰기 질의 향상'을 증폭 약속으로 거는 내신·수행평가 글쓰기 코치 — 베끼는 AI가 아니라, 들켜도 떳떳하고 점수가 오르게 가르치는 합법 코치."**

**레드팀 반영 핵심 수정:** 원본은 "입증 가능한 성적 상승(아웃컴)"을 단일 1차 축으로 박았다. 레드팀 비판 #2가 타당하다 — 우리가 파는 행위(수행평가 수정 첨삭)는 본인 리서치상 효과크기가 가장 작은 영역(g=0.55, 수정엔 작고 전이엔 큼)이고, 학부모가 사는 "등급"은 표시광고법상 못 쓴다. 따라서 **위계를 역전**한다.

- **1차 약속 = 합법성·과정증거** (정책이 만든 확실한 수요, 효과크기 리스크 없음, 측정 깔끔)
- **증폭 약속 = 아웃컴(글쓰기 질 향상)** (효과크기 리스크 있음 → 실측 후 부착)
- **해자 = 종단 데이터 독점** (입증요건이 아님 — 레드팀 #1 반영)

**결정 척추(grilling 2026-06-08 확정, 위배 금지):**
- 제품: 중·고 수행평가/내신 5영역 글쓰기 AI 첨삭. 데모는 정적 EPO→라이브 채점 완성.
- 락(고정): 브랜드 + OS 셸 멤버십(blue+lemon). 그 외 전부 증거로 도전 가능.
- 승리 형태: **아웃컴 무어를 종단 데이터 해자로 재정의**. 범용 ChatGPT를 원시능력으로 못 이긴다는 전제 유지.
- 증명 표준: 2-tier. ①지금 = 블라인드 before/after 루브릭 재채점(proxy). ②지속 = 기관 준실험. **단, 둘 다 출시 전 게이트 통과 전까지 "1등 로드맵"이 아닌 "1등 가설".**
- 최대 리스크(확정): 웜 기관 없음(SNT에듀 사용 불가). 기관 콜드스타트 전제.
- 컴플라이언스: Korea-first. 개인정보보호법 14세 + 표시광고법 효과광고가 게이트.

## 2. ⛳ 체크포인트 — 아침에 검토·방향전환할 것 (레드팀 핵심 의사결정 통합)

### 🚦 출시 전 반드시 통과해야 할 3개 게이트 (레드팀 최종판정 반영, 신규)

이 문서는 아래 3게이트를 넘기 전까지 **"1등 로드맵"이 아니라 "1등 가설"**이다. 게이트 미통과 시 해당 축을 발표·집행하지 않는다.

| 게이트 | 통과 기준 | 미통과 시 |
|---|---|---|
| **G1. N% 실측** | 파일럿 n=30으로 블라인드 재채점 효과크기 측정. N이 두 자릿수(≥10%대) + p<0.05 | 아웃컴 메시지를 메인에서 내리고 합법성 단일축 |
| **G2. 광고법 사전 의견서** | 광고법 전문 로펌 의견서 또는 공정위 비조치의견 1건 확보(proxy가 "합리적·객관적 근거"인지) | 효과 주장 전면 보류, '가성비(즉시·일관·저비용)' 사실광고만 |
| **G3. 채널 1개 실증** | DTC 무료→유료 전환율 ≥ 가설치(예 5%) **또는** 강사-시딩 1곳 before/after 자산화 성공 중 하나 | 부트스트랩 경로 전면 재설계 |

### (a) 부트스트랩 경로 권고안 — **트리플 트랙, 단 강사-시딩을 공동 1순위로 재승격** (레드팀 #3 반영)

원본은 강사-시딩을 3순위로 강등했으나 레드팀 #3이 타당하다. 강사는 **CAC≈0 + 웜 신뢰 + before/after 자연 생산**의 3박자를 가진 유일 채널이고, "이해상충"은 오판 — 강사는 첨삭 노동에서 해방되고 모객 자산을 얻으므로 **이해일치**다. 강등 근거였던 소프트랩 제외요건은 본인이 "법인 분리로 해결 가능"이라 적었다. 따라서:

- **공동 1순위 A — DTC (만14세+ 헤비 가구):** 직접 경쟁자 전원이 학교 조달에 몰려 DTC 아웃컴 제품이 빔. 수요 기반(96.5% AI 사용·0점 불안) 존재. **단, 콜드스타트 리스크를 §11 1행으로 격상**(레드팀 #3) — "96.5%"는 수요가 아니라 무료 ChatGPT 대체재 점유율이며 월 4만원 전환 증거는 0.
- **공동 1순위 B — 강사-시딩 (개별 자율 논술강사):** CAC≈0·웜신뢰·이해일치·before/after 자연 생산. **소프트랩과 동시 추구 시 사업법인 분리 필수**(제외요건⑥). 상위 프랜차이즈(대교=투블럭 락) 회피.
- **2순위 — KERIS 소프트랩 (콜드 기관):** **단, 기대치를 "무료 학교 레퍼런스 + 정성 효과 점검"으로 낮춤**(레드팀 #4). 준실험을 KERIS가 대신해주지 않는다 — 준실험은 별도 설계.

**경로 선택은 G3로 결정한다.** A·B 중 먼저 실증되는 쪽이 1차 엔진. 두 다리가 다 모래 위라는 레드팀 지적을 받아, 둘을 병렬로 검증하고 살아남는 쪽에 집중.

### (b) 아웃컴 무어가 성립 안 할 경우의 대안 — **백업이 아니라 1차로 승격** (레드팀 #2 반영)

원본은 "합법성 무어"를 백업으로 뒀으나 레드팀 #2가 타당하다 — 아웃컴이 깨지면 축을 통째로 바꾸는 건 "전략이 두 개라는 자백"이다. 그래서 **위계를 역전**해 합법성을 1차로 올린다.

- **1차 = 합법성·과정증거 (정책이 만든 확실 수요):** 교육부 가이드라인이 'AI 무수정 제출=0점'을 못박았으므로, **'AI 사용 투명 로그' + '직접 썼다는 드래프트 이력'**을 신뢰 자산으로 판다. 효과크기 리스크 없음. 경쟁사 미보유. 96.5% 0점 불안에 직격.
- **증폭 = 아웃컴(글쓰기 질 향상):** g=0.55, I²=81%로 설계가 좌우 ([PMC10351274](https://pmc.ncbi.nlm.nih.gov/articles/PMC10351274/)). G1 실측 후 N이 충분할 때만 메인 메시지로 부착.
- **표시광고법 폴백:** proxy 불인정 시(선례 0건) '가성비(즉시·일관·저비용)' 사실광고. 강사 회당 4~5만 vs 월구독, 강사 4일 지연 vs 즉시.

### (c) 리서치가 전략 척추와 충돌하는 지점 (사용자 판단 필요)

1. **'5영역 수행평가 한정' vs 전이 학습설계 — 충돌(강화됨).** 메타분석상 효과크기는 '수정'에서 작고 '전이'에서 크다. **레드팀 #2·#6 반영:** 우리가 파는 게 바로 효과 최저 영역이므로, 제품 내부를 반드시 **전이 유도 루프**로 설계. 정책상 집필단계 차단으로 "집 연습→교실 성적" 전이가 더 어려워질 수 있음 → 전이 가설을 파일럿에서 직접 측정.
2. **'성적 오른다' 메시지 vs 표시광고법 — 부분 충돌.** "성적/등급 상승"은 금지. **"표준 루브릭 기준 글쓰기 질 N% 향상(블라인드 2인 재채점, n=X, p<0.05)"**만 가능. 단 §2-b대로 이건 증폭이지 1차 약속 아님.
3. **DTC 14세 게이트 vs 중1 포함 — 마찰.** **'만 14세 이상(중2~고3)' 우선 출시**, 중1은 보호자 동의 별도 트랙. 사용자 승인 필요.
4. **'화이트스페이스 = 성적주장 0개'는 부재 증명이라 단정 불가.** "실재하나 위험한 영역"으로 표현. KEEwi가 700곳 데이터로 종단 성과 선발표 시 차별점 잠식 — **속도가 생명, 단 G2 통과 전 부실 입증 강행 금지**(레드팀 #7).
5. **(신규) 단품 1등 vs OS 부품 — 충돌(레드팀 #5).** 3중4중 좁힘 끝 SAM이 단품 사업으로 성립하는지 bottom-up 숫자 부재. 출시 전 통계청 마이크로데이터로 SAM 확정. 작으면 **OS all-access Pass의 전환·리텐션 드라이버**로 정직하게 포지셔닝 전환. 어느 쪽인지 사용자 확정 필요.
6. **(신규) 내신 원점수 확보 경로 — 미검증(레드팀 #4).** 학교 절대평가 원점수를 외부 SW가 합법 수취하는 경로는 사실상 '교사 자발 제공+동의'뿐 → 자기선택 편향으로 Tier2 인과성 약화. Tier2가 생각보다 멀다는 것을 직시.

## 3. 포지셔닝 & 웨지: 왜 ChatGPT도 강사도 아닌 우리인가

> 레드팀이 "가장 강한 점 #1"로 인정한 섹션. 통찰은 유지하되 1차 약속을 합법성으로 재배치.

### 화이트스페이스 좌표 (교집합)

```
                  챗봇(범용 LLM)이 못 하는 것
                  · 루브릭 정렬 진단
                  · 수행평가 채점포인트 점수 예측
                  · 집필 前 합법 연습 + 사용 투명 로그(2026 가이드라인 순응)
                         ╲          ╱
                          ╲   ★    ╱   ← Pullim
                           ╲ 교집합╱
                         ╱          ╲
   강사가 못 하는 것     ╱            ╲
   · 즉시성(4일→즉시)
   · 일관성(채점자 편차 제거)
   · 저비용(회당 5만→월구독)
   · 24시간 확장
```

**근거:**
- 범용 LLM은 **생성기이지 평가기준 대비 진단·점수 예측기가 아님**. 학생 사용은 '그대로/요약·재진술/틀세우기/종합/도약' 5유형 전부 생성 모드 ([이윤빈 2024](https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART003114677)). 2026 가이드라인으로 **집필 단계 사용 차단** ([교육부 2025.12.23](https://www.korea.kr/briefing/pressReleaseView.do?newsId=156736923)).
- 강사 첨삭은 비싸고(시간당 평균 4.5만, 회당 4~5만) 느리고(지연) 표현 첨삭에 머묾 ([숨고](https://soomgo.com/prices/논술), [orbi](https://orbi.kr/0001610829)).
- 96.5%가 이미 LLM을 쓰는 시장에서 **'또 하나의 글쓰기 AI'는 죽는다. '들켜도 떳떳하게 가르치는 합법 코치'만 산다.**
- **레드팀 #6 반영 — 부정행위 낙인 리스크:** 교사가 '외부 AI 첨삭=부정행위'로 간주할 권한 미확인. 최악 시 풀림 사용 자체가 낙인 → DTC 학부모 기피. **완화: '사용 투명 로그'를 교사 채널에 무료 배포해 풀림을 '부정행위 도구'가 아니라 '정직성 증빙 도구'로 사회적 승인 선점.**

## 4. 증명 프로그램 (2-tier) + 해자 재정의

### 🛡️ 해자 재정의 — "입증요건"이 아니라 "종단 데이터 독점" (레드팀 #1 반영)

레드팀 #1이 타당하다. "입증이 어려워 경쟁자가 못 따라온다"는 논리는 입증이 **우리에게도 어렵다**는 뜻이고, 자본 10배인 메가스터디·투블럭에게 똑같이 높은 벽은 해자가 아니다. 따라서:

- **진짜 해자 = 학생별 글쓰기→성적 종단 궤적 데이터 독점.** 한 번 입증한 사실이 아니라 시간이 갈수록 쌓이는 데이터. 후발주자가 자본으로 못 산다.
- **입증요건은 해자가 아니라 "진입 자격증"** — 통과해야 시장에 발 들이지만, 그 자체가 방어선은 아님.
- **"매주 자동 생산"의 인건비 환상 제거(레드팀 #1c):** 현직 교사 2인 독립 채점을 매주 n=50은 인건비 블랙홀. → **LLM-as-judge + 인간 보정(n의 일부만)** 하이브리드로 비용 현실화. 단 그 방법론 자체의 신뢰도(인간-LLM QWK)를 **먼저** 검증한 뒤 사용.

### Tier 1 — Proxy: 블라인드 before/after 루브릭 재채점

## 5. 제품/기능 로드맵 (기능깊이 약점 정조준)

전제: 데모는 라이브 채점 완성. '6지표 첨삭'(KEEwi 동일)으로는 못 이긴다. **1차 약속(합법성)을 받치는 것을 최우선으로** 만든다(위계 역전 반영).

### 만든다 (Build) — 우선순위 위계 역전

1. **[1차] AI 사용 투명 로그 자동 생성:** "어떤 프롬프트·어떻게 반영" 기록 → 2026 가이드라인 요건 충족. **'들켜도 떳떳한' 합법성 = 1차 판매 명제.** 효과크기 리스크 없음. 교사 채널 무료배포로 낙인 리스크 차단(§3).
2. **[1차] 드래프트 이력·고쳐쓰기 로그(과정 증거):** '학생이 직접 썼다'를 강사·학부모에게 신뢰 자산으로. 경쟁사 미보유.
3. **[증폭] 블라인드 재채점 엔진(하이브리드):** LLM-judge + 인간 보정. G1 실측 인프라. **인건비 현실화 설계(레드팀 #1c).**
4. **[증폭] 성장 그래프 대시보드:** 5영역 시계열 + N% 향상 배지. **단 G1 통과 후 노출.**
5. **[해자] 전이 유도 학습 루프:** '이번 피드백을 다음 글에 스스로 적용'하는 발판. 메타분석상 전이 효과크기가 커서 무어를 키움 + 종단 데이터 축적.
6. **빈 화면 공포 제거 발판:** 단계별 발판 + 즉시 피드백 = retention 핵심.

### 죽인다 / 보류 (Kill / Defer)

> 레드팀이 "가장 강한 점 #3"으로 인정한 규율. 유지.

- **교사용 채점 자동화 진입 — 죽인다.** 정부 무료(채움아이 2027 전교)+선점(KEEwi·클리포). **단 §3 낙인완화용 '사용 로그'는 교사에 무료배포(채점 자동화와 구분).**
- **수능 논술 영역 확장 — 보류.** 정책 동력 없음.
- **단순 점수 채점만 제공 — 죽인다.** 공교육 무료 내재화 + 콴다 가격 압박.
- **범용 생성/초안 기능 — 죽인다.** LLM 정면충돌 + 2026 가이드라인 위반.

## 9. 컴플라이언스 플랜 (14세 + 표시광고법)

### 9.1 표시광고법 — "성적 오른다"를 합법적으로 말하기

- **G2 게이트 신설:** 광고 게재 전 **광고법 전문 로펌 사전 의견서 또는 공정위 비조치의견 1건 확보**(레드팀 #1b). "선례 0건"을 "선례 1건(우리)"으로 만드는 것이 진짜 선점.
- **실증 패키지 선완비:** 블라인드 재채점·QWK·표본·방법 문서화. 공정위 15일 요청에 즉시 대응.
- **표현 가이드:**
  - ✅ "표준 루브릭 기준 글쓰기 질 N% 향상(블라인드 재채점, n=X, p<0.05)" (G1·G2 후, 증폭용)

  - ❌ "성적이 오릅니다" / "등급 상승 보장" / 자가보고 설문만 근거(디지털대성 직격)
- **후기 사용 시:** 추천·보증 심사지침(2024.12) — 실제·검증 사례 + 대가 명시, 생성AI 가상인물 명시.

- **연령 정책:** 만 14세 이상(중2~고3) 본인 동의 우선. 중1(만13세)은 법정대리인 동의 별도 트랙. 위반 시 5년/5천만원·매출 3% ([제22조의2](https://casenote.kr/법령/개인정보_보호법/제22조의2)).

**최종 한 줄:** 한국에 1등이 없는 빈 카테고리는 맞다 — 그러나 빈칸은 "성적 증명"이 아니라 **"들켜도 떳떳한 합법 코치"**다. 정책이 만든 0점 불안(96.5%)은 확실한 수요이고, 효과크기가 작은 아웃컴은 증폭일 뿐이다. 진짜 해자는 입증요건이 아니라 **종단 데이터 독점**이며, 부트스트랩은 두 다리(DTC·강사-시딩)를 병렬로 검증해 살아남는 쪽에 건다. **이 문서는 3개 게이트(N% 실측·광고법 의견서·채널 1개 실증)를 넘기 전까지 "1등 로드맵"이 아니라 "1등 가설"이며, Pullim은 누구보다 빨리 — 그러나 부실 입증으로 자멸하지 않게 — 증명해야 산다.**
